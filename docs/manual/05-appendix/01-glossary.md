# Glossary

Key terms and definitions used throughout this project.


| Term | Definition |
|------|------------|
| **DialogueLog** | Parsed dialogue log. |
| **GeneratedFile** | A generated file. |
| **GeneratedMCP** | Result of MCP generation. |
| **GeneratorEngine** | Engine for generating MCP servers from patterns. |
| **Input** | An input parameter for a pattern. |
| **LearningEngine** | Engine for learning from build outcomes and adjusting scores. |
| **LogParser** | Parser for markdown dialogue logs. |
| **MinnaSync** | Synchronize patterns with Minna Memory for cross-session persistence. |
| **OutcomeStats** | Statistics for build outcomes. |
| **Output** | An output produced by a pattern. |
| **Pattern** | A reproducible workflow pattern extracted from logs. |
| **PatternExtractor** | Extracts workflow patterns from parsed dialogue logs. |
| **PatternStore** | File-based store for patterns using YAML. |
| **ScoreAdjustment** | Adjustment to apply to a pattern score. |
| **ScoreWeights** | Configurable weights for pattern scoring. |
| **ScoredPattern** | A pattern with buildability scores. |
| **ScoringEngine** | Engine for scoring patterns on buildability. |
| **Session** | A logical session of related turns. |
| **Step** | A workflow step in a pattern. |
| **ToolSpec** | Specification for a generated tool. |
| **Trigger** | A trigger condition for a pattern. |
| **Turn** | A single turn in a dialogue (user or assistant). |
| **ValidationError** | A validation error. |
| **ValidationResult** | Result of validation. |
| **Validator** | Validator for generated MCP code. |
| **spawn** | Meta-MCP server - analyzes patterns in AI agent sessions and generates new MCP servers from them |
